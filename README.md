# Scribble

## Overview

Scribble is a desktop application designed to help users automate text input and extract text from images. It combines adjustable typing simulation with built-in Optical Character Recognition (OCR) to support productivity, accessibility, and efficient workflows.

This tool is intended for legitimate use cases such as drafting, transcription, accessibility support, and converting image-based text into editable formats.

---

## Features

### Automated Typing

* Simulates human-like typing behavior
* Adjustable typing speed
* Optional delays between keystrokes for more natural pacing
* Useful for demonstrations, accessibility needs, and repetitive text entry

### OCR (Image to Text)

* Extracts text from images
* Supports common image formats such as PNG and JPG
* Converts screenshots, scanned documents, and photos into editable text

### Customization

* Control typing speed and delay
* Configure behavior to match your workflow
* Lightweight and simple to use

---

## Use Cases

* Accessibility assistance for users who have difficulty typing
* Transcribing text from images or screenshots
* Drafting documents with automated input support
* Testing or demonstrating typing-related applications

## How to Use

### Typing Simulation

1. Open the application
2. Enter or paste the text you want to be typed
3. Adjust typing speed and delay settings
4. Select the target application or window
5. Start the typing simulation

### OCR (Image to Text)

1. Open the OCR feature in the application
2. Upload or drag and drop an image
3. Click "Extract Text"
4. Copy or export the generated text
